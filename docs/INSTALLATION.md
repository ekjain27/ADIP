# Installation Guide

## Requirements

Python 3.11+

Node.js 20+

Git

Modern Web Browser

---

## Clone

git clone <repository>

---

## Backend

cd backend

pip install -r requirements.txt

python main.py

---

## Frontend

cd frontend

npm install

npm run dev

---

## Tests

pytest

---

## Dashboard

Open:

http://localhost:5173

---

Installation Completed.
