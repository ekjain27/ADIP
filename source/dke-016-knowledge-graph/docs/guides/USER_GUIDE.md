# User Guide

## Purpose

This guide explains how an end user interacts with Project-1 through the frontend UX modules.

## UX-001 Enterprise Dashboard

Use the dashboard to view platform status, release readiness, module health, and high-level decision intelligence state.

## UX-002 Decision Workspace

Use the Decision Workspace to review demo decisions, create new demo decisions, load previous decisions, and inspect evaluation/ranking consistency.

## UX-003 Knowledge Graph & Provenance Explorer

Use the graph explorer to inspect graph entities, provenance paths, edge details, timeline state, and graph filters.

## UX-004 Explainability & Analytics Dashboard

Use this dashboard to generate explanations, refresh analytics, inspect timeline information, and review decision transparency indicators.

## UX-005 Documentation / Patent / Research Generator

Use this section to generate or preview documentation, patent, and research materials based on implemented generator workflows.

## UX-006 Administration & User Management

Use this section to manage demo users, remove users, refresh user state, and inspect audit-log activity.

## UX-007 Enterprise Platform Integration

Use this section to manage demo integrations, webhook state, refresh behavior, and integration test visibility.

## Important Scope Limit

Project-1 does not include enterprise data onboarding, file upload, external datasets, or live production API onboarding.
